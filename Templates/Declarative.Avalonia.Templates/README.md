# Declarative.Avalonia Templates

This package contains the project template for `dotnet new avalonia-declarative`.

## Install

```bash
dotnet new install Declarative.Avalonia.Templates
```

## Use

```bash
dotnet new avalonia-declarative -n MyApp
```

The generated project is based on the declarative component sample and includes:

- Avalonia desktop bootstrap code
- Fluent theme setup
- `Avalonia.Markup.Declarative`
- a small DI-backed component example
